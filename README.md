  # Public Assets

  This repository hosts static binary assets used
  during Docker image builds for Einpix applications.

  ## Contents

  | Asset | Version | Description |
  |---|---|---|
  | google-chrome-stable.deb | 134.0.6998.88-1 | Google Chrome for PDF rendering |

  ## Usage

  Assets are distributed via GitHub Releases and
  referenced directly in Dockerfiles by their stable
  release URL.
